# Deep-Learning-Tabanli-Stok-Raf-Takip-Projesi
Teknolojiler: Python, MobileNetV3, Tiny-GAPNet. Perakende sektöründe stok takibi ve raf düzeni için derin öğrenme modelleri geliştirildi. Düşük donanımlı cihazlarda yüksek performans için MobileNetV3 ve Tiny-GAPNet mimarileri kullanıldı.Karmaşık görsellerde ürün tespiti yapılarak insan hataları minimize edildi.
